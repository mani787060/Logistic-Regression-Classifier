# Logistic Regression Classifier

## 📌 Project Overview

This project demonstrates the complete implementation of **Logistic Regression**, one of the most widely used algorithms for **binary classification** in Machine Learning.

Using datasets generated with **`make_classification`** and **`make_blobs`**, this notebook explains the complete machine learning workflow—from generating synthetic datasets to training, evaluating, and visualizing a Logistic Regression classifier. It helps build a strong understanding of linear decision boundaries, probability-based classification, and model evaluation.

---

## 🎯 Objectives

* Understand the fundamentals of Logistic Regression
* Learn how binary classification works
* Generate and analyze synthetic classification datasets
* Train a Logistic Regression model
* Evaluate model performance using multiple classification metrics
* Visualize decision boundaries and predictions

---

## 📂 Dataset

**Datasets Used:**

### `make_classification`

A synthetic dataset generated using Scikit-Learn for binary classification. It allows complete control over the number of samples, features, informative variables, and class separation.

### `make_blobs`

A synthetic clustering dataset used to visualize how Logistic Regression separates different classes using a linear decision boundary.

## 🔍 Key Observations

* Logistic Regression performs well on linearly separable datasets.
* The model predicts probabilities using the Sigmoid function.
* Feature scaling can improve optimization and model performance.
* Evaluation metrics like Precision, Recall, and ROC-AUC provide better insights than accuracy alone.
* Decision boundary visualization helps understand how the classifier separates different classes.
